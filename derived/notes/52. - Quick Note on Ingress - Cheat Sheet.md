## Core Idea

Use ingress for outside routing, and make sure you are reading docs for the correct ingress project.

## What Ingress Does

- traffic comes in
- ingress routes it to the right service
- client and server can both sit behind it

## Important Project Distinction

- use: `kubernetes/ingress-nginx`
- do not confuse it with: `nginxinc/kubernetes-ingress`
